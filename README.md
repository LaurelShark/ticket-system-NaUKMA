"# ticket-system-NaUKMA" \
Цей проект знаходиться на стадії розробки, після узгодження з викладачем, будуть завантажені файли з початковим кодом.

Для запуску нашої системи потрібно:
-	Встановити локальний сервер (наприклад OpenServer)
-	Розмістити проект в папці OpenServer\domains\localhost
-	Записати в змінну середовища користувача PATH шлях до php.exe, якщо вона не встановлена
-	Відкрити командний рядок та перейти в кореневий каталог проекту
-	Виконати команду composer install
-	Після налаштування composer, потрібно запустити локальний сервер – програму Open Server x64.exe
-	В браузері ввести точку входу http://localhost/BASIC-PR/basic/web/index.php?r=login%2Findex
